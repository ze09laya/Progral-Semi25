* 3 Formularios intermedios (complejidad o con 2 o 3 tablas)
* 3 Formularios anvanzados (alta complejidad o 3 o mas tablas)
* 10 reportes (salida de informacion o representacion de informacion al usuario)
* Menus y accesos rapidos, barras de herramientas, cuadros de dialogo
* Creatividad e innovacion.
